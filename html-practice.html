Inline-level Elements

Inline elements are elements whose horizontal width is determined by the width of the content they contain. The <strong> element and the <em> element are both examples of inline elements.

Inline elements do not take up their own line of horizontal space. Manually add a <br> to move to a new line.

Adding an image

Informative Images

For informative images, alternative text should clearly and concisely describe the subject matter of the image, without referring to the image itself. For example, do not write “Image of a Gidget the dog, Secret Life of Pets” but "Gidget the dog, Secret Life of Pets mascot."


<img src="gidget.png" alt="Gidget the dog, Secret Life of Pets mascot." width="200"/>

The div element

The HTML Content Division element <div> acts as a container to structure a webpage into separate components for individual styling. We can specify attributes to adjust its size, color, position, or other features. Typically, with CSS

Adding color

We can use HTML to modify the text color of elements such as <p> or <h1>

This is green text.

<p style="color:green;">This is green text.></p

The color of a border is modified by using the <style> attribute and the <border> property:

<img src="https://html.sammy-codes.com/images/small-profile.jpeg" style="border: 2px solid blue"/>



<img src="https://html.sammy-codes.com/images/small-profile.jpeg" style="border: 2px dashed lightseagreen"/>



Colours can be added in hexadecimal values too: like #0000FF (blue), #40E0D0 (turquoise), or #C0C0C0 (silver)

Setup & Format the HTML project

The first step is to define the <!DOCTYPE html>. This declaration tells the browser which type of HTML is being used as there are multiple versions of HTML. In this declaration, html specifies the current web standard of HTML, which is HTML5.

The second step is to define the <html> element. This tells the browser that all content it contains is intended to be read as HTML. We also add the <lang> element, which specifies the language of the webpage

The third step is to define the <head> element. This creates a section to include information about our web document and that's primarily used by browsers and search engines to interpret the content of the page (SEO positioning). We then add the <title> and the <meta> elements. The <meta charset="utf-8"> specifies the document’s character set to UTF-8, a unicode format that supports a majority of characters from a wide variety of written languages.

Next step is to add a favicon to the website with HTML. Recommended: use simple and high-contrast images only. A good site to create them is favicon.cc A favicon is a small image that is located in the browser tab to the left of a webpage's title. Create a folder called "images" and save the favicon image there.

Then we define the <body> element. If we add style="margin: 0;" we will remove the margin surronding the page. If we add style="text-align:center;" we can align the text.

Finally, we add a <footer> section. For checking the metadata try The Open Graph protocol and check the full Twitter Card markup specification.. To test the Webpage Metadata, use the Open Graph Debugger or try the Twitter Card validator, Facebook Sharing Debugger, LinkedIn Post Inspector, or Pinterest Rich Pins Validator
